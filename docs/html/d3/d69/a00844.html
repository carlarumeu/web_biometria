Descripción detallada

Constructs a 64 bit two's-complement integer, given its low and high 32 bit values as signed integers.

See the from* functions below for more convenient ways of constructing Longs. @exports Long for representing a 64 bit two's-complement integer value.

Parámetros
{number}low The low (signed) 32 bits of the long
{number}high The high (signed) 32 bits of the long
{boolean=}unsigned Whether unsigned or not, defaults to signed @constructor
