My\+SQL2 project is a continuation of \href{https://github.com/sidorares/nodejs-mysql-native}{\texttt{ My\+SQL-\/\+Native}}. Protocol parser code was rewritten from scratch and api changed to match popular \href{https://github.com/mysqljs/mysql}{\texttt{ Node My\+SQL}}. My\+SQL2 team is working together with \href{https://github.com/mysqljs/mysql}{\texttt{ Node My\+SQL}} team to factor out shared code and move it under \href{https://github.com/mysqljs}{\texttt{ mysqljs}} organization.

My\+SQL2 is mostly API compatible with \href{https://github.com/mysqljs/mysql}{\texttt{ Node My\+SQL}} and supports majority of features. My\+SQL2 also offers these additional features\+:


\begin{DoxyItemize}
\item Faster / Better Performance
\item \href{https://sidorares.github.io/node-mysql2/docs/documentation/prepared-statements}{\texttt{ Prepared Statements}}
\item My\+SQL Binary Log Protocol
\item \href{https://sidorares.github.io/node-mysql2/docs/documentation/mysql-server}{\texttt{ My\+SQL Server}}
\item Extended support for Encoding and Collation
\item \href{https://sidorares.github.io/node-mysql2/docs/documentation/promise-wrapper}{\texttt{ Promise Wrapper}}
\item Compression
\item SSL and \href{https://sidorares.github.io/node-mysql2/docs/documentation/authentication-switch}{\texttt{ Authentication Switch}}
\item \href{https://sidorares.github.io/node-mysql2/docs/documentation/extras}{\texttt{ Custom Streams}}
\item \href{https://sidorares.github.io/node-mysql2/docs\#using-connection-pools}{\texttt{ Pooling}}
\end{DoxyItemize}\hypertarget{a00943_autotoc_md1700}{}\doxysubsubsection{\texorpdfstring{Installation}{Installation}}\label{a00943_autotoc_md1700}
