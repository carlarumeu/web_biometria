Define a data property on an object. Will fall back to assignment in an engine without descriptors.

The three {\ttfamily non\texorpdfstring{$\ast$}{*}} argument can also be passed {\ttfamily null}, which will use the existing state if available.

The {\ttfamily loose} argument will mean that if you attempt to set a non-\/normal data property, in an environment without descriptor support, it will fall back to normal assignment.\hypertarget{a00874_autotoc_md411}{}\doxysubsection{\texorpdfstring{Usage}{Usage}}\label{a00874_autotoc_md411}
