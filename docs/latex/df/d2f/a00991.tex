Unpipes all destinations from a given stream. With stream 2+, this is equivalent to {\ttfamily stream.\+unpipe()}. When used with streams 1 style streams (typically Node.\+js 0.\+8 and below), this module attempts to undo the actions done in {\ttfamily stream.\+pipe(dest)}.\hypertarget{a00991_autotoc_md2498}{}\doxysubsection{\texorpdfstring{License}{License}}\label{a00991_autotoc_md2498}
