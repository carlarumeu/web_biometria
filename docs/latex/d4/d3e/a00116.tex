Transfer the file at the given {\ttfamily path}. 

Automatically sets the {\itshape Content-\/\+Type} response header field. The callback {\ttfamily callback(err)} is invoked when the transfer is complete or when an error occurs. Be sure to check {\ttfamily res.\+headers\+Sent} if you wish to attempt responding, as the header and some data may have already been transferred.

Options\+:


\begin{DoxyItemize}
\item {\ttfamily max\+Age} defaulting to 0 (can be string converted by {\ttfamily ms})
\item {\ttfamily root} root directory for relative filenames
\item {\ttfamily headers} object of headers to serve with file
\item {\ttfamily dotfiles} serve dotfiles, defaulting to false; can be {\ttfamily "{}allow"{}} to send them
\end{DoxyItemize}

Other options are passed along to {\ttfamily send}.

Examples\+:

Options\+:


\begin{DoxyItemize}
\item {\ttfamily max\+Age} defaulting to 0 (can be string converted by {\ttfamily ms})

\item {\ttfamily headers} object of headers to serve with file
\item {\ttfamily dotfiles} serve dotfiles, defaulting to false; can be {\ttfamily "{}allow"{}} to send them
\end{DoxyItemize}
