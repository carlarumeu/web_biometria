\hypertarget{a00939_autotoc_md1652}{}\doxysubsection{\texorpdfstring{Features}{Features}}\label{a00939_autotoc_md1652}

\begin{DoxyItemize}
\item Works both in \href{https://nodejs.org}{\texttt{ node}} and in the browser.
\item If a number is supplied to {\ttfamily ms}, a string with a unit is returned.
\item If a string that contains the number is supplied, it returns it as a number (e.\+g.\+: it returns {\ttfamily 100} for `\textquotesingle{}100'\`{}).
\item If you pass a string with a number and a valid unit, the number of equivalent ms is returned.
\end{DoxyItemize}\hypertarget{a00939_autotoc_md1653}{}\doxysubsection{\texorpdfstring{Caught a bug?}{Caught a bug?}}\label{a00939_autotoc_md1653}
